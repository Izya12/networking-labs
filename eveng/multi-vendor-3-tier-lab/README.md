# Multi Vendor 3 Tier Lab
Multi vendor lab based on a 3 tier topology.
* Vendors are Cisco, Juniper and Arista.
* Core is based on OSPF.
* Aggregation uses vPC.
* LACP downlinks from aggregation to each access switch.
